# Encyclopédie des Arthropodes - Walfarm Family

Site gratuit hébergé avec GitHub Pages.

## Fichiers principaux

- `index.html` : page d'accueil
- `style.css` : apparence du site
- `script.js` : catégories, recherche et premières fiches
- `images/` : logo et bannière
- `cartes/` : futures images des cartes
- `fiches/` : futures fiches détaillées
